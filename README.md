# Weather App

Modern ve responsive bir hava durumu uygulaması. React ile geliştirildi. Kullanıcı şehir arayabilir, mevcut konumuna göre hava durumunu görebilir ve 5 günlük tahmini inceleyebilir.

## Özellikler

- Şehir adına göre hava durumu arama
- Kullanıcının konumuna göre otomatik hava durumu
- 5 günlük hava tahmini
- Hava durumuna göre dinamik arka plan
- Dark / Light mode
- Responsive tasarım

## Kullanılan Teknolojiler

- React
- JavaScript
- CSS
- OpenWeather API
